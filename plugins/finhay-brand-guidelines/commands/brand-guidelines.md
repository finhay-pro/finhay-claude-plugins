# Finhay Brand Guidelines

## Overview

This skill provides Finhay Vietnam's complete brand identity system for creating brand-consistent artifacts across all platforms and formats.

**When to use:**
- Creating presentations, documents, or web content
- Applying brand colors (App/Web UI or Marketing)
- Using logos correctly
- Writing brand-aligned copy and messaging
- Building UI components or marketing materials
- Ensuring visual consistency across products
- Creating Vietnamese language content

**Keywords**: Finhay, brand guidelines, design system, Vietnamese fintech, brand colors, typography, Inter font, visual identity, logo, brand voice, marketing

### Color Ratio Guidelines (Marketing)

**Standard Design (60-30-10):**
- 60% Primary color
- 30% Secondary color
- 10% Accent (5-7% orange, 3-5% others)

**Partner Content:**
- Finhay channels: 70% Finhay / 30% Partner
- Joint: 50% / 50%
- Partner channels: Min 20% Finhay

**Events:**
- Regular events: 80% Brand / 20% Event
- Cultural/Traditional: 60% Brand / 40% Event

## Logo

> **CRITICAL: NEVER recreate, redraw, or generate the Finhay logo.** Always use the exact PNG files from `assets/logos/`. The logo contains precise gradients, shapes, and proportions that cannot be accurately reproduced. Any attempt to draw, code, or generate an approximation of the logo is strictly prohibited.

### Logo Usage Rule

**ALWAYS embed/reference the actual logo files. NEVER:**
- Draw the logo using code (SVG, CSS, Canvas, etc.)
- Describe the logo for AI image generation
- Use emoji or unicode symbols as logo substitutes
- Create simplified versions of the logo
- Generate or recreate any part of the logo

**INSTEAD, always use the provided PNG files** by embedding or referencing them directly in your output.

### Logo Structure
- **Symbol**: Two overlapping organic leaf shapes at 45° angle
- **Wordmark**: "Finhay" in Inter Bold
- **Formats**: Horizontal (primary) and Vertical

### Logo Meaning
- Two leaves = Growth potential, nurturing investments
- 45° angle = Balanced, sustainable growth
- Green = Vitality, trust, prosperity

## Brand Voice

### Voice Pillars

1. **Thông thái - Hướng dẫn** (Wise & Guiding) - Data-backed, logical
2. **Gần gũi - Đời thường** (Approachable) - Everyday language
3. **Truyền cảm hứng** (Inspiring) - Positive, motivating
4. **Hài hước tinh tế** (Witty) - Subtle, clever humor

### Quick Do's & Don'ts

**Do:**
- Use simple, clear language
- Stay positive and confident
- Back claims with data
- Use subtle humor appropriately
- Include disclaimers for investments

**Don't:**
- Use unexplained jargon
- Be too casual or sarcastic
- Use fear-inducing language
- Use ALL CAPS or excessive !!!
- Make absolute promises

See [BRAND_VOICE.md](BRAND_VOICE.md) for complete guidelines.

## Vietnamese Content

### Key Rules

- Always use diacritics (đầu tư, not dau tu)
- No abbreviations in formal content
- Date format: DD/MM/YYYY
- Currency: 1.000.000đ (period for thousands)
- Phone: (024) 1900 292 99

See [VIETNAMESE_CONTENT.md](VIETNAMESE_CONTENT.md) for complete guidelines.

## Design Style

### Semi-Flat Design

Finhay uses Semi-Flat Design combining:
1. Flat design simplicity
2. Subtle gradients for depth
3. Noise texture for organic feel
4. Rounded corners for friendliness
5. Glass effects for modern touch

### Key Graphics: Finlight

The Finlight system derives from the logo's leaf symbol:
- **Mảng Finlight** - Solid leaf shapes
- **Đường Finlight** - Line-based outlines
- **Finlight Hội Tụ** - Converging lines
- **Finlight Lan Toả** - Radiating patterns

See [KEY_GRAPHICS.md](KEY_GRAPHICS.md) for complete guidelines.

## Quick Checklist

When creating branded content, ensure:

- [ ] Correct color system for context (App/Web vs Marketing)
- [ ] Inter font family used
- [ ] Logo used correctly (position, clear space, colors)
- [ ] Typography hierarchy maintained
- [ ] 4.5:1 contrast minimum (WCAG AA)
- [ ] Vietnamese text renders correctly with diacritics
- [ ] Brand voice aligned (wise, approachable, inspiring, witty)
- [ ] Disclaimers included for financial content
- [ ] Semi-Flat Design style followed
- [ ] Consistent with other Finhay materials
