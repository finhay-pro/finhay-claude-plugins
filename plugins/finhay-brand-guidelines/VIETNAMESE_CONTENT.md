# Vietnamese Content Guidelines

Standards for creating Vietnamese language content that aligns with Finhay's brand voice and professional standards.

## Spelling & Grammar Rules

### General Rules

| Rule | ✓ Correct | ✗ Incorrect |
|------|-----------|-------------|
| No abbreviations | "Thứ Hai", "Thành phố" | "T2", "TP" |
| Always use diacritics | "Đầu tư", "Tài chính" | "Dau tu", "Tai chinh" |
| No spelling errors | "khoản", "chứng khoán" | "khoan", "chung khoan" |
| Avoid ALL CAPS | "Thông báo quan trọng" | "THÔNG BÁO QUAN TRỌNG" |
| Capitalize after periods | "Đầu tư. Bạn có thể..." | "Đầu tư. bạn có thể..." |
| Single space between words | "đầu tư tài chính" | "đầu  tư  tài  chính" |

### Capitalization Rules

| Context | Rule | Example |
|---------|------|---------|
| Sentence start | Capitalize first letter | "Đầu tư thông minh..." |
| After period | Capitalize first letter | "...thành công. Bạn có thể..." |
| Product names | Capitalize as branded | "HayBond", "Finhay" |
| Proper nouns | Capitalize | "Việt Nam", "Hà Nội" |
| Common nouns | Lowercase | "nhà đầu tư", "quỹ đầu tư" |

### Punctuation Rules

| Rule | ✓ Correct | ✗ Incorrect |
|------|-----------|-------------|
| No space before punctuation | "thành công." | "thành công ." |
| No space before punctuation | "Xin chào," | "Xin chào ," |
| Single space after punctuation | "Xong. Tiếp theo" | "Xong.Tiếp theo" |
| Limit exclamation marks | "Chúc mừng!" | "Chúc mừng!!!" |
| No period after URLs/emails | "finhay.com.vn" | "finhay.com.vn." |
| Use Vietnamese quotation marks | "đầu tư" or 'đầu tư' | «đầu tư» |

### Common Punctuation

| Mark | Usage | Example |
|------|-------|---------|
| `.` (period) | End of sentence | "Giao dịch thành công." |
| `,` (comma) | Pause, list separation | "Quỹ A, Quỹ B, và Quỹ C" |
| `:` (colon) | Introduction, explanation | "Lưu ý: Đầu tư có rủi ro" |
| `;` (semicolon) | Related clauses | "Lợi nhuận cao; rủi ro cũng cao" |
| `?` (question mark) | Questions | "Bạn muốn đầu tư?" |
| `!` (exclamation) | Excitement (use sparingly) | "Chúc mừng!" |
| `—` (em dash) | Emphasis, aside | "Finhay — ứng dụng đầu tư hàng đầu" |
| `...` (ellipsis) | Trailing off, continuation | "Còn nhiều hơn thế..." |

### Text Layout Rules

| Rule | Description |
|------|-------------|
| Compound words | Keep compound words together; don't break across lines |
| Widows | Avoid single words alone on the last line of paragraphs |
| Orphans | Avoid single lines at the top of new pages/columns |
| Line breaks | Break at logical phrase boundaries, not mid-word |
| Paragraph length | Keep paragraphs short (3-5 sentences max for digital) |

**Example - Line Breaking:**
```
✓ Correct:
Đầu tư thông minh
giúp bạn tích lũy tài sản.

✗ Incorrect:
Đầu tư thông
minh giúp bạn tích lũy tài sản.
```

## Formatting Standards

### Phone Numbers

**Format**: `(Mã vùng) Dãy số`

| ✓ Correct | ✗ Incorrect |
|-----------|-------------|
| (024) 1900 292 99 | 024.1900.292.99 |
| (028) 3823 4567 | 028-3823-4567 |
| 0901 234 567 | 0901.234.567 |

**Mobile numbers**: Group as `0XXX XXX XXX`
**Hotlines**: Keep original format if branded (e.g., "1900 292 99")

### Addresses

**Format**: 
```
Tầng [số], Tòa nhà [tên], số [số] [đường/phố], 
Phường/Xã [tên], Quận/Huyện [tên], [Thành phố/Tỉnh]
```

**Example:**
```
✓ Correct:
Tầng 8, Tòa nhà Capital Building, số 58 phố Kim Mã, 
Phường Ba Đình, Quận Ba Đình, TP Hà Nội

✗ Incorrect:
T8, Capital Building, 58 Kim Mã, Ba Đình, HN
```

**Abbreviations (use sparingly):**
- TP = Thành phố (acceptable in addresses)
- P. = Phường (acceptable)
- Q. = Quận (acceptable)

### Email Addresses

| Rule | Example |
|------|---------|
| Use hyperlink format | [support@finhay.com.vn](mailto:support@finhay.com.vn) |
| No capitalization | support@finhay.com.vn |
| No trailing period | finhay.com.vn ✓ / finhay.com.vn. ✗ |

```
✓ Correct: support@finhay.com.vn
✗ Incorrect: Support@Finhay.Com.Vn
✗ Incorrect: support@finhay.com.vn.
```

### Website URLs

| Rule | Example |
|------|---------|
| Hyperlink format preferred | [finhay.com.vn](https://finhay.com.vn) |
| No "http://" prefix in display | finhay.com.vn |
| No "www." unless required | finhay.com.vn (not www.finhay.com.vn) |
| No trailing slash | finhay.com.vn (not finhay.com.vn/) |

```
✓ Correct: finhay.com.vn
✓ Correct: vnsc.vn
✗ Incorrect: http://www.finhay.com.vn/
```

### Date Format

**Vietnamese Standard**: `DD/MM/YYYY` or written form

| Format | Example | Usage |
|--------|---------|-------|
| Numeric | 29/12/2025 | Forms, data, technical |
| Written short | 29 tháng 12, 2025 | Marketing, editorial |
| Written full | Ngày 29 tháng 12 năm 2025 | Formal documents |

```
✓ Correct: 29/12/2025
✗ Incorrect: 12/29/2025 (US format)
✗ Incorrect: 2025-12-29 (ISO format, unless technical)
```

**Day of Week:**
- Thứ Hai, Thứ Ba, Thứ Tư, Thứ Năm, Thứ Sáu, Thứ Bảy, Chủ Nhật
- Capitalize "Thứ" and day name

### Time Format

**Vietnamese Standard**: 24-hour or 12-hour with clarification

| Format | Example |
|--------|---------|
| 24-hour | 14:30, 09:00 |
| 12-hour | 2:30 chiều, 9:00 sáng |

**Time zones**: Use "giờ Việt Nam" or "GMT+7" if needed

### Currency

**Vietnamese Đồng (VND)**

| Format | Example | Usage |
|--------|---------|-------|
| Symbol after | 1.000.000đ | Common, informal |
| Code after | 1.000.000 VND | Formal, international |
| Written | Một triệu đồng | Contracts, formal |

**Formatting rules:**
- Thousands separator: `.` (period)
- Decimal separator: `,` (comma) - rarely used for VND
- No space between number and đ symbol
- Space before VND code

```
✓ Correct: 1.000.000đ
✓ Correct: 1.000.000 VND
✗ Incorrect: 1,000,000 VND (US format)
✗ Incorrect: 1.000.000 đ (space before đ)
```

**Large numbers:**
| Amount | Written |
|--------|---------|
| 1.000.000đ | 1 triệu đồng |
| 1.000.000.000đ | 1 tỷ đồng |
| 1.500.000đ | 1,5 triệu đồng |

### Numbers

| Rule | Example |
|------|---------|
| Thousands separator | 1.000.000 (period) |
| Decimal separator | 4,5% (comma) |
| Spell small numbers in prose | "hai chiếc lá", "ba năm" |
| Use numerals for data/stats | "tăng 15%", "3 triệu người dùng" |

### Percentages

```
✓ Correct: 4,5%/năm
✓ Correct: tăng 15%
✗ Incorrect: 4.5%/năm
```

## Vietnamese Typography

### Required Character Support

The Vietnamese alphabet requires full support for:

**Base consonants:**
```
b c d đ g h k l m n p q r s t v x
```

**Vowels with diacritical marks:**
```
a ă â e ê i o ô ơ u ư y
```

### Diacritical Marks (Dấu)

All tone marks must be properly supported and rendered:

| Base | Sắc (´) | Huyền (`) | Hỏi (?) | Ngã (~) | Nặng (.) |
|------|---------|-----------|---------|---------|----------|
| a | á | à | ả | ã | ạ |
| ă | ắ | ằ | ẳ | ẵ | ặ |
| â | ấ | ầ | ẩ | ẫ | ậ |
| e | é | è | ẻ | ẽ | ẹ |
| ê | ế | ề | ể | ễ | ệ |
| i | í | ì | ỉ | ĩ | ị |
| o | ó | ò | ỏ | õ | ọ |
| ô | ố | ồ | ổ | ỗ | ộ |
| ơ | ớ | ờ | ở | ỡ | ợ |
| u | ú | ù | ủ | ũ | ụ |
| ư | ứ | ừ | ử | ữ | ự |
| y | ý | ỳ | ỷ | ỹ | ỵ |

### Font Requirements

| Requirement | Specification |
|-------------|---------------|
| Primary font | Inter (full Vietnamese support ✓) |
| Secondary font | MTD Adventures (decorative use only) |
| Fallback | System fonts with Vietnamese support |
| Testing | Always preview Vietnamese text before publishing |

**Fonts to avoid:**
- Fonts without Vietnamese diacritic support
- Decorative fonts for body text
- Fonts with poor diacritic positioning

### Font Testing Checklist

Before using any font, test these phrases:
```
Đầu tư thông minh với Finhay
Lợi nhuận ổn định, tích lũy bền vững
Quỹ mở, chứng chỉ quỹ, trái phiếu
VIỆT NAM - FINHAY - ĐẦU TƯ
```

## Content Localization

### Cultural Considerations

| Aspect | Guideline |
|--------|-----------|
| **Colors** | Green = prosperity, growth (positive in Vietnamese culture) |
| **Numbers** | 8, 9 = lucky; 4 = sometimes avoided (sounds like "death" in Chinese) |
| **Holidays** | Respect Tết, Mid-Autumn, National Day, etc. |
| **Family** | Family references are positive and relatable |
| **Age** | Respect for elders; avoid age-related jokes |
| **Tone** | Formal but friendly; avoid slang |

### Lucky/Unlucky Associations

| Element | Association | Usage |
|---------|-------------|-------|
| 8 | Prosperity | Good for prices, dates |
| 9 | Longevity | Good for promotions |
| 6 | Smooth | Generally positive |
| 4 | Avoid | Don't emphasize in pricing |
| Red | Luck, prosperity | Use for celebrations |
| Green | Growth, money | Core brand color |
| Yellow/Gold | Wealth, royalty | Premium features |

### Pronoun Usage

Vietnamese pronouns convey relationship and respect. Use appropriately:

| Pronoun | Usage | Context |
|---------|-------|---------|
| **Bạn** | "You" (friendly) | Default for customers, peer-level |
| **Quý khách** | "You" (formal) | Very formal, customer service |
| **Chúng tôi** | "We" (company) | Finhay as an organization |
| **Chúng ta** | "We" (inclusive) | Finhay + customer together |
| **Mình** | "I/We" (casual) | Very informal, social media only |

**Examples:**
```
Formal: "Quý khách vui lòng kiểm tra thông tin"
Standard: "Bạn có thể bắt đầu đầu tư ngay hôm nay"
Inclusive: "Chúng ta cùng xây dựng tương lai tài chính"
```

## Writing Style for Vietnamese

### Sentence Structure

| Preference | Example |
|------------|---------|
| Active voice | "Finhay giúp bạn đầu tư" ✓ |
| Passive voice (avoid) | "Việc đầu tư được hỗ trợ bởi Finhay" ✗ |
| Short sentences | "Bắt đầu từ 100.000đ. Dễ dàng. An toàn." |
| Complex sentences (limit) | Keep to one main idea per sentence |

### Paragraph Structure

- **Digital content**: 2-4 sentences per paragraph
- **Print content**: 4-6 sentences per paragraph
- **One idea per paragraph**
- **Topic sentence first**

### Bullet Points & Lists

When using lists in Vietnamese:

```
✓ Good:
Lợi ích của Finhay:
• Đầu tư từ 100.000đ
• Giao dịch 24/7
• Đội ngũ hỗ trợ chuyên nghiệp

✗ Avoid:
Lợi ích của Finhay:
• đầu tư từ 100.000đ (no capitalization)
• Giao dịch 24/7. (inconsistent punctuation)
• Đội ngũ hỗ trợ chuyên nghiệp!!!
```

### Headlines & Titles

| Style | Example | Usage |
|-------|---------|-------|
| Sentence case | "Đầu tư thông minh với Finhay" | Default |
| Title case | Not common in Vietnamese | Avoid |
| ALL CAPS | "ĐẦU TƯ THÔNG MINH" | Avoid (except logos) |
